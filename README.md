# 智能IT客服
### 介紹
此project主要是架設一個智能IT客服系統，用戶能夠透過與chatbot詢問問題，得到問題的解法，若資料庫目前沒有問題的解法，chatbot能夠幫助客戶完成報案，並將資料送至後端資料庫，客服人員能夠操作前端網頁去查看、編輯、刪除報案資料，如下圖:<br>
