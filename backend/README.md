# Backend

use Fastify, Typescript and MongoDB

## Available Scripts

```
npm run build
```

Build the app for production to the out folder.

```
npm run start
```

Run the backend app with node.

## Run MongoDB with docker

Make sure ```docker``` is installed

```
docker run -d -p 27017:27017 mongo
```
